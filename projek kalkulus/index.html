Understanding Quadratic Functions

Definition

A quadratic function is a polynomial function of degree 2, written in the form:

f(x) = ax² + bx + c

where:

  • a ≠ 0 (determines opening direction and width)
  • b affects the axis of symmetry
  • c is the y-intercept
